## Console ChatBot (C++)

Консольный чат-бот "вопрос–ответ" с возможностью:
- добавления вопроса и ответа
- поиска ответа по вопросу
- редактирования ответа
- редактирования вопроса
- удаления записи
- вывода списка всех вопросов
- сохранения базы в файл
- загрузки базы из файла

## Запуск
Проект написан на C++ и собирается в Visual Studio.

## Структура проекта
- main.cpp — меню и сценарии работы
- ChatBot.cpp/.h — операции над базой (поиск/добавление/редактирование/удаление)
- file_io.cpp/.h — загрузка/сохранение в файл
- utils.cpp/.h — вспомогательные функции ввода
